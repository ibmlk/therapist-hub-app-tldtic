## How to Update This Changelog

When making changes:

1. Add entries under `[Unreleased]` section
2. Use appropriate subsections (Added, Changed, Fixed, etc.)
3. Be clear and concise
4. Include issue/PR numbers when applicable
5. When releasing, move unreleased items to a new version section

Example:
```markdown
### Added
- New therapist verification system (#123)
- Push notification support (#124)

### Fixed
- Booking time zone issue (#125)
- Profile image upload bug (#126)
```
